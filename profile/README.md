## Imajeure

**An indie software studio. Some of it's useful. Some of it's just fun.**

We're a small studio building software across a few different shapes —
tools people rely on, and things made purely for the joy of them.

### What we make

- **[Sirenade](https://sirenade.app)** — reminders that escalate
  (push → email → SMS → call) until they're acknowledged. The useful kind.
- **[Quotetoo](https://quotetoo.com)** — a home for words worth keeping.
  Your words, your legacy.

### Open source

- **[mcp-stdio-bridge](https://github.com/imajeure/mcp-stdio-bridge)** — wrap any
  stdio MCP server as a self-healing streamable-HTTP endpoint
  (automatic child respawn + systemd watchdog).
- **[mcp-shell](https://github.com/imajeure/mcp-shell)** — token-gated MCP shell
  server: bearer auth, default-deny command allowlist, localhost by default.
